## Project Title

**Landing Page for Students Registration**

## Project Description

This project is a simple web-based student registration portal developed as part of the SEN 304 course. It enables students to register, upload a passport photograph, and log in using their registered credentials. The application uses HTML, CSS, and JavaScript, with browser Local Storage for data persistence.

## Features

- Student registration
- Student login
- Password confirmation validation
- Passport photograph upload with preview
- Nigerian States and Local Government Areas (LGAs) dropdown
- Student dashboard after successful login
- Responsive Glassmorphism user interface
- Local Storage for saving user information

## Technologies Used

- HTML5
- CSS3
- JavaScript (ES6)
- Local Storage API
